# 1. Diagrama de Flujo del Proceso

Puedes describirlo así en tu documento (o dibujarlo con flechas):

Inicio -> Pantalla de Login -> Ingreso de credenciales -> Validación -> Pantalla de Inventario -> Selección de Producto -> Agregar al Carrito -> Ver Carrito -> Checkout -> Ingreso de datos -> Confirmación -> Logout -> Fin.

# 2. Listado de Páginas y Elementos Clave

Para SauceDemo, las páginas son:

|**Página**|**Elementos Clave**|
|---|---|
|**Login Page**|Input Username, Input Password, Button Login.|
|**Inventory Page**|Botones "Add to cart", Dropdown de filtros, Ícono de carrito.|
|**Cart Page**|Lista de productos agregados, Botón "Checkout".|
|**Checkout Step One**|Inputs de First Name, Last Name y Zip Code.|
|**Checkout Complete**|Mensaje de éxito, Botón "Back Home".|
